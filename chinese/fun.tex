\cvsection{趣事~}

\vspace{4.0mm}

\begin{cvitems}
  \item 我在很多地方使用 ``liblaf'' 作为用户名或昵称.
        它来源于 ``Life blooms like a flower''.
        每个单词的首字母合起来就是 ``lblaf''.
        为了便于发音, ``Life'' 一词取前两个字母并将 ``l'' 改为 ``li'', 于是得到了 ``liblaf''.
  \item 你知道为什么我总是戴着兜帽吗?
        因为我曾经被一只从树上掉下来的毛毛虫砸中过脑袋!
        % 想象一下, 那毛茸茸的小家伙从天而降, 砸在我的头上, 让我差点把头发都掉光了!
        % 所以现在我保险起见, 一出门就得戴上兜帽, 防范于未然!
\end{cvitems}
